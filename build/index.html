This message was authored / outboxed at beep:boop compute beep:boop, but the sender wanted to send it later for some reason, which is why you're receiving it now.

Sometimes I would be enjoying a moment, and would want to remember it later. "But when later?" I would ask myself - so I made this to decide that for me.

I use this with schedule send in Gmail to email things to myself or others.
